# 🌸 快速开始
1. 拉取项目[https://gitee.com/wenroudoubinanhai/czh-uni.git](https://github.com/15139467313/big-screen-love.git)；
2. 通过vscode导入项目；
3. 命令行执行 npm install 安装依赖
4. 命令行执行 npm run serve 启动
5. 命令行执行 npm run  build:prod 打包
